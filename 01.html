安装Node.js

下载Node.js

  • 1.查看：计算机-右键-属性-系统属性-查看自己电脑系统类型，是32或是64位操作系统。
  • 2.下载32位或64位安装包、Mac或Linux下载请看这里。
  • 3.安装目录最好是默认，如果您相当有信心换到其他盘符也行。默认安装路径："C:\Program Files\nodejs\"
  • 4.完成安装。开始-附件-命令提示符 【打开命令提示符】 输入 【node-v】 查看版本号。

    环境配置。

    在"C:\Program Files\nodejs\"文件下新建node_cache和node_global文件夹。
  • 1.在开始-附件中-命令提示符-右键用管理员身份运行输入下面两行代码 
    npm config set prefix "C:\Program Files\nodejs\node_global
    npm config set cache "C:\Program Files\nodejs\node_cache

    重点

    桌面-计算机-右键-属性-高级系统设置-高级-环境变量。
  • 1.在【系统变量下面】中新建变量名：【PATH】变量值：【C:\Program Files\nodejs\;C:\Program Files\nodejs\node_cache】
  • 2.在【系统变量下面】中新建变量名：【NODE_PATH】变量值：【C:\Program Files\nodejs\node_global\node_modules】
  • 3.在【用户变量上面】中新建或编辑变量名：【Path】 变量值：【C:\Program Files\nodejs\node_global】
