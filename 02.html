配置GitBook

  • 1.在开始-附件-命令提示符-右键用管理员运行 打开输入
    npm install gitbook-cli -g
  • 2.找一个盘。 比如：在我的E盘中 新建文件夹-重命名为【02】文件夹 。
    • 1.开始-附件-命令提示符-直接打开 不用管理员运行 进入E盘的02文件夹下。
    • 2.命令是：两个步骤输入【e:】和【cd 02】。如图
    • 3.安装gitbook 输入命令 
      gitbook init

      构建目录

      此时该文件夹下会生成俩文件【README.md】和【SUMMARY.md】
  • README.md: 书的介绍文字，如前言、简介，在章节中也可做为章节的简介。
  • SUMMARY.md: 定制书籍的章节结构和顺序。
    • 目录的语法如下：* [显示的目录名](文件夹/文件名#标题id)。
